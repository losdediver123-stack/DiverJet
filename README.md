# DiverJet

Web de catálogo de viajes low cost hecha con HTML + CSS, sin base de datos.

## Diseño
El diseño visual de esta web está inspirado en una propuesta creada en **Stitch (Google)** y adaptada para este proyecto.

## Reparto de trabajo
- **Sergio:** estructura principal del HTML, navegación y tarjetas de destinos.
- **Hugo:** textos de ofertas, bloque de contacto y revisión visual móvil.
- **IA:** ajuste de la cuadrícula responsive con media queries y apoyo en detalles visuales.

### Prompt usado para la parte de IA
> "Dame una versión más elegante y moderna de la web de DiverJet, con estética parecida a Stitch, manteniendo HTML y CSS simples, clases en español y responsive básico."
