# DiverJet

Web de catálogo de viajes low cost hecha con HTML + CSS, sin base de datos.

## Diseño
- El diseño está basado en la estética de Stitch (Google), adaptado a una versión simple para este proyecto.
- Se ha evitado usar APIs externas: el HTML solo enlaza a `styles.css`.
- Las imágenes no vienen de internet, están creadas y guardadas en la carpeta `fotos/`.
