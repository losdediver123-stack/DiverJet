# DiverJet

Web de catálogo de viajes low cost hecha con HTML + CSS, sin base de datos.

## Reparto de trabajo
- **Sergio:** estructura principal del HTML, navegación y tarjetas de destinos.
- **Hugo:** textos de ofertas, bloque de contacto y revisión visual móvil.
- **IA:** ajuste de la cuadrícula responsive con media queries.

### Prompt usado para la parte de IA
> "Haz una cuadrícula responsive sencilla para tarjetas de viajes, con CSS claro, pocos nombres de clase y en español."
